Info


This plugin has everything you need to easily and quickly show all reviews about you.

Very fast start

  • Install plugin via npm or bower
  • Include plugin files in your app

JS methods to control for the widget

The widget has a concise and simple methods:

  1. Start
  2. Stop
  3. Next
  4. Add
  5. LoadSlide

Minimalist and comfortable structure of the slide

  • author
    • name
    • url
    • avatar(url)
  • company
    • name
    • url
  • quote

Project use best practices

  • Clean code base
  • Unit tests
  • Install via npm or bower
  • Minimalist and concise plugin API
  • Use git and open source
  • MIT License
