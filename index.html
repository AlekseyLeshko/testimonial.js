Testimonial.js

Show all testimonial letters about your business!

Install plugin

with bower 

$ bower install testimonial --save-dev

with npm 

$ npm i -D testimonial

Use

Include files testimonial.min.css and testimonial.min.js. And do not forget jQuery.

Example

The plugin is written using js and css. Uses the library jquery. Plugin is run, there are tests and generally very good plugin!
